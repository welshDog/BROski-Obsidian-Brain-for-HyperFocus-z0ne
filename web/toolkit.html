Start Focus

Begin a tracked focus session. Intent, duration, project, tags, and difficulty dial all wired in.
POST /focus/start
🛑
End Focus
LIVE
Close an active session. Log actual minutes, mood (1–10), and reflection notes. Awards XP + coins.
POST /focus/end

LIVE
Check if a session is currently active, the current intent, elapsed time, and session ID.
