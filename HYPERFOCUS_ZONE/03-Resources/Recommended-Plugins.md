## ⚙️ QuickAdd Setup (After Install)

1. Settings → QuickAdd → Add Choice
2. Name: "Brain Dump" → Type: Template
3. Template: `99-Templates/Brain-Dump.md`
4. Folder: `00-Inbox`
5. Set hotkey: `Ctrl+Q`

Now `Ctrl+Q` → instant brain dump → auto-filed to Inbox. 🧠⚡

---

## ⚙️ Homepage Setup (After Install)

1. Settings → Homepage
2. Set "Homepage" to: `Hub/Dashboard`
3. Toggle "Open on startup" → ON
4. Now every time you open Obsidian → Dashboard appears
