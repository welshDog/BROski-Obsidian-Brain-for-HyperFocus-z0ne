# 🧠 Truth Pack — Front Door
> One place to sync Brain A (Obsidian) + Brain B (Repos) + Brain C (Live Systems).
> Rule: Proof beats memory.

## Read Order (always)
1. [SYSTEM_STATUS.md](./SYSTEM_STATUS.md) — what’s true right now (proof links only)
2. [KNOWN_ISSUES.md](./KNOWN_ISSUES.md) — what’s broken (symptom → proof → fix)
3. [DECISIONS.md](./DECISIONS.md) — why we did it (stop re-debating)
4. [RUNBOOKS.md](./RUNBOOKS.md) — how to do the key ops
5. [INTEGRATIONS.md](./INTEGRATIONS.md) — where things connect (secret locations, never values)

## Update Ritual (2 minutes)
- End of session: update SYSTEM_STATUS (3 bullets max)
- If something broke: add it to KNOWN_ISSUES with one proof link
- If we made a call: add a single decision to DECISIONS
