## What We're Building

Hyperfocus Zone is the world's first neurodivergent-first autonomous AI infrastructure platform — built in Llanelli, Wales, by a solo ADHD/dyslexic/autistic founder.

The platform is live. It runs a 30-container Docker stack (HyperCode-V2.4), a 35-agent AI swarm across 5 Docker networks, a published npm agent framework (@w3lshdog/hyper-agent), a Web3 NFT productivity game (BROskiPets), a gamified course platform (Hyper-Vibe Coding Course), and a second brain vault (BROski-Obsidian-Brain) with 8 live modules, a 192-node knowledge graph, and 89 hero-named agent skills.

This is not a prototype. This is infrastructure.

## The Problem We Solve

Neurodiverse people — ADHD, dyslexic, autistic — represent approximately 20% of the UK population. Existing productivity and AI tools are built for neurotypical users. They overwhelm, they don't adapt, they don't chunk tasks. The result: massive untapped human potential.

Hyperfocus Zone flips this. Every architectural decision — from the HyperSplit task decomposition engine to the BROski Coin reward system to the ADHD-safe UI patterns — is built around how neurodivergent brains actually work.

## Why Sovereign AI Funding Now

The stack is built. The agents are running. The bottleneck is compute.

All LLM inference currently exits the UK to US-based API endpoints. The local Ollama container is CPU-only — max 7B models at 4 tokens/sec, unusable for multi-agent fine-tuning workloads. Commercial H100 access via AWS/GCP/Azure costs £2.5M–£4M for 1M GPU hours — impossible for an early-stage solo founder.

Sovereign compute access via AIRR (Isambard-AI) or Dawn unlocks three R&D workstreams that cannot be completed any other way:

1. **Fine-tuning a domain-adapted agent behaviour model** (7B–13B, UK-sovereign inference)
2. **Creating the world's first neurodivergent productivity AI dataset** (100k+ examples, openly published)
3. **Running the first rigorous multi-agent orchestration benchmark suite** for accessibility-focused AI

All three outputs will be published as open UK AI assets.
