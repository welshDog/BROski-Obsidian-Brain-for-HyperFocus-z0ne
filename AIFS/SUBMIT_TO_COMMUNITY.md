# 🚀 AIFS Community Submission Guide

> Step-by-step. Do these in order. 30 minutes total.

---

## Step 1 — Fork + PR to punkpeye/awesome-mcp-servers (5 mins)

```bash

# 2. Clone your fork
git clone https://github.com/YOUR_USERNAME/awesome-mcp-servers
cd awesome-mcp-servers

# 3. Add AIFS to the README
# Find the "Developer Tools" or "Productivity" section
# Add this line:
```


```bash
# 4. Commit + PR
git add README.md
git commit -m "Add AIFS — AI File System folder contract MCP server"
git push
# Then open a PR on GitHub
```

---

## Step 2 — Post on dev.to (10 mins)

1. Go to https://dev.to/new
2. Copy the full post from `AIFS/BLOG_POST.md`
3. Set tags: `ai`, `python`, `mcp`, `opensource`, `devtools`
4. Set cover image: use `architecture_diagram.png` from the repo root
5. Hit **Publish**
6. Share the URL in the AIFS repo README

---

## Step 3 — Submit to mcp.so (5 mins)

1. Go to https://mcp.so/submit (or similar submit form)
2. Fill in:
   - **Name:** AIFS — AI File System
   - **Repo:** https://github.com/welshDog/BROski-Obsidian-Brain-for-HyperFocus-z0ne
   - **Description:** Folder contract protocol for AI agents. Per-folder permissions, real-time enforcement, Ed25519 signing, live dashboard, public registry.
   - **Tags:** governance, filesystem, mcp, python, developer-tools
